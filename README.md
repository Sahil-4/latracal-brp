## Get the Repository

1. Clone repository

```bash
git clone
```

## Install & Run (Local)

1. Start Frontend

```bash
cd frontend
```

2. Install dependencies

```bash
npm install
```

3. Start dev server

```bash
npm run dev
```

1. Start Backend

```bash
cd backend
```

2. Install dependencies

```bash
npm install
```

3. Start compile

```bash
npm run compile
```

4. Start dev server

```bash
npm run dev
```
